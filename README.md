---

## 💡 Название проекта: `Product Checker`

---

## 🎯 **Цель проекта**

Создать систему, которая позволяет пользователю определить **оригинальность товара по его штрихкоду** (EAN-13), используя:

- 📲 Ручной ввод кода
- 📸 Сканирование с камеры устройства
- 🖼 Загрузку фото/изображения штрихкода

---

## 🧩 **Ключевые компоненты**

### 1. **REST API на Go**

С помощью `Gin` и `GORM`, сервер предоставляет следующие возможности:

- 🔹 CRUD операции с товарами
- 🔹 Проверка подлинности товара по EAN-13
- 🔹 Хранение истории проверок (опционально)
- 🔹 Возможность подключения мобильных клиентов (Android/iOS)

---

### 2. **Функции API**

| Метод | Эндпоинт | Описание |
| --- | --- | --- |
| `POST` | `/products` | Создание товара |
| `GET` | `/products` | Получение всех товаров |
| `GET` | `/products/:id` | Получение одного товара |
| `PUT` | `/products/:id` | Обновление товара |
| `DELETE` | `/products/:id` | Удаление товара |
| `POST` | `/check-barcode` | Проверка подлинности по штрихкоду |

---

### 3. **Проверка оригинальности**

Алгоритм реализован вручную — контрольная цифра рассчитывается по стандарту EAN-13:

- Сумма цифр с чётных позиций × 3
- Сумма цифр с нечётных позиций
- Общая сумма, и сравнение контрольной цифры (13-я цифра)

Если контрольная цифра не совпадает — код **некорректный/поддельный**.

---

### 4. **Веб-интерфейс (на Go + HTML + JS)**

Пользователь может:

- 🔍 Ввести код вручную
- 📷 Нажать “Сканировать” (открыть камеру через HTML5)
- 📂 Загрузить изображение (будет отправлено на сервер для обработки, OCR — в будущем)

Интерфейс будет простым, с результатом вида:

```
✔️ Товар оригинальный
ИЛИ
❌ Подделка

```
